# Heterodyna-SDR 
Nowoczesne narzędzie do analizy i skanowania sygnałów radiowych w oparciu o urządzenia SDR (Software Defined Radio).

Heterodyna SDR to rozbudowana aplikacja desktopowa, zaprojektowana z myślą o pasjonatach RF, inżynierach oraz specjalistach zajmujących się analizą widma. Program umożliwia monitorowanie, dekodowanie oraz wizualizację sygnałów w czasie rzeczywistym.

Główne funkcje

📡 Skanowanie pasma RF w czasie rzeczywistym

🌊 Waterfall (widmo + aktywność sygnałów)

🎯 Detekcja sygnałów (manualna i automatyczna)

📊 Analiza częstotliwości i mocy sygnału

🔊 Obsługa różnych modulacji (AM / FM / NFM / WFM i inne)

📁 Zapis danych i logów (np. JSONL / IQ / audio)

🧠 Tryby inteligentnej analizy (AI / heurystyka – w rozwoju)

🛠 Integracja z RTL-SDR i kompatybilnymi urządzeniami


🧪 Zastosowania:

analiza pasma radiowego

nasłuch i eksperymenty SDR

edukacja i badania RF

wykrywanie emisji (np. FPV, transmisje cyfrowe)

diagnostyka środowiska radiowego

⚙️ Technologie

.NET / C# (desktop GUI)

integracja z SDR (RTL-SDR / DLL / sterowniki)

przetwarzanie sygnałów (DSP)

wizualizacja danych w czasie rzeczywistym

⚠️ Informacje

Projekt rozwijany eksperymentalnie. Część funkcji może być w trakcie implementacji lub testów.

## ⚙️ Wymagania

- Windows 10/11 64 Bit
- RTL-SDR (np. RTL-SDR V3)
- .NET Runtime (wymagany)

## 🔧 Sterowniki

Aplikacja posiada wbudowany mechanizm instalacji sterowników (Zadig / WinUSB).

➡️ W większości przypadków wystarczy użyć opcji instalacji w aplikacji.

⚠️ Jeśli instalacja automatyczna nie zadziała:
- użyj Zadig ręcznie
- ustaw sterownik WinUSB dla urządzenia RTL-SDR
